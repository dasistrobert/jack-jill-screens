## How to Use

1. Open your Figma file after running the plugin scripts
2. Select a frame (e.g., '000037 — Jobs Inbox')
3. Drag the matching optimized PNG onto it
4. The image will fill the frame at 1440x900

### Bulk method (faster):
1. Select all optimized PNGs in Finder
2. Drag them all onto the Figma canvas at once
3. Resize each to fit its matching frame
4. Or use the Figma plugin 'Insert Big Image' for large screenshots
